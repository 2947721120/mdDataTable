mdtRow

Description

Representing a row which should be placed inside mdt-table element directive.

Dependencies

  • mdtTable

Usage

as element:
<mdt-row
       [table-row-id="{string|integer}"]>
</mdt-row>

Parameters

ParamTypeDetails
tableRowId
(optional)
stringinteger

when set table will have a uniqe id. In case of deleting a row will give back this id.
